# ExLite

[![npm downloads](https://img.shields.io/npm/dm/exlite.svg)](https://www.npmjs.com/package/exlite)
[![npm version](https://img.shields.io/npm/v/exlite.svg)](https://www.npmjs.com/package/exlite)

**ExLite** is a comprehensive utility library for Express.js that simplifies backend and API development. It provides a set of reusable helper functions and features to make building APIs easier, more efficient, and more maintainable.

## Key Features:

- **Simplified Route and Controller Management**: Pre-built helpers streamline how you manage routes and controllers, allowing for more organized and readable code.
- **Integrated Error Handling**: Provides consistent error handling across all routes and middleware, ensuring that unexpected issues are captured and managed gracefully.
- **Error Handling Wrappers**: Automatic error-catching with easy-to-use wrappers, reducing the need for manual error handling in every route.
- **Customizable Response Formatting**: Offers a simple way to format API responses consistently, ensuring that your output structure remains uniform across all endpoints.
- **Dependency Injection Support**: Built-in integration with [`tsyringe`](https://github.com/microsoft/tsyringe) for flexible and scalable dependency injection.
- **Custom Error Classes**: Supports custom error classes for fine-grained control over error handling, making it easier to manage various error scenarios.
- **Efficient HTTP Management**: Helps in managing HTTP status codes and responses in a clean and predictable way.

## Ecosystem Libraries

ExLite is part of a larger ecosystem of tools designed to simplify Express.js development:

- [`exlite`](https://github.com/aashishpanchal/exlite/tree/main/packages/exlite): Core library with all essential features for Express.js applications.
- [`@exlite/serve`](https://github.com/aashishpanchal/exlite/tree/main/packages/serve): A utility designed to serve static web applications in Express.js environments.
